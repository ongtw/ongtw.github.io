Animation Demo written in Python and Kivy

This animation demo was done to learn the Python programming language proper, and also the Kivy framework for UI.
To blog about development experience working with Python, Kivy and pygame for graphics and audio.
To mention quirks/gotcha's and other technical stuff learned through experimentation (can't find detailed docs).

Some stuff to blog:

  • Kivy kv file: widget hierarchy traversal, event/properties bindings
  • kv file: widgets to call python code behind (e.g. to callback on_press event continuously)
  • python code to access kv file widgets
  • remove specific instructions (e.g. sprites created on the fly) from Kivy canvas
  • device independent graphics (Windows PC vs retina MacBook vs iOS iPhone)
  • drawing (local/global coords), clipping
  • pygame mixer quirk: make sure mp3 is (re-)sampled to 44100 hz to play correctly, or it will be slower than usual (even at 48Khz!)
